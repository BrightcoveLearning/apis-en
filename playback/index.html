--- title: Playback API parent: Home description: '

The Playback API is used to fetch video and playlist data from Video Cloud. It is typically used to get video data to a player on a web page or in a mobile app. The Playback API is low-latency API intended for client-side use in fetching video or playlist data from web pages or mobile apps. It is not a general media management API to use in integrating Video Cloud with your CMS or other systems - for that, you should use the CMS API.

Authentication for the Playback API requires a policy key, which you can obtain using the Policy API or from a player in your account.
