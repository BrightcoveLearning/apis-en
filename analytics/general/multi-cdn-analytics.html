Introduction

Video Cloud accounts can be enabled to use multiple CDNs, include the Brightcove house CDN and BYO CDNs. The Analytics API has a special dimensions cdn_type and bandwidth_category that can be queried together with the account dimension to obtain data on how many bytes were delivered through the multiple CDNs.
