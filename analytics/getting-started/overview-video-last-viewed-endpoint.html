Introduction

The last date viewed endpoint provides a quick way of finding out when one or more of the videos in your account were last viewed.

Endpoint

The endpoints for last date viewed requests are as follows:

Get results for an account

GET

https://data.api.brightcove.com/historical/media-insights/v1/all/account/{accountId}/lastview

Query parameters

  • pageSize (integer, up to 1000, default: 1000 - how many results to return)
  • page (integer, default: 1 - which results page to return)

Get results for multiple videos by ids

https://data.api.brightcove.com/historical/media-insights/v1/all/accounts/{accountId}/lastview/bulk/media-ids

Request body

Up to 1000 media ids are allowed per request.

Authentication

All requests to the Analytics API are authenticated via Brightcove's OAuth service.

Authentication is based on an access token passed in an Authorization header:

    Authorization: Bearer {your token}

You can get client credentials for the Analytics API and use those to retrieve temporary access tokens - see the OAuth Overview for an explanation of different approaches you can use to obtain client credentials and access tokens, and Managing API Credentials for information on obtaining client credentials through Studio
