Live Analytics Endpoints

In addition to the live_stream dimension covered in this topic, There are also special endpoints for retrieving additional analytics data for Brightcove Live streaming jobs. See the Analytics API Reference for details.

{% include s-apis-general/analytics/url-params.html %}

Fields available

The following fields can be returned for the dimension.

  • bytes_delivered - total bytes delivered for live streams, including the player and all associated media
  • origin_bytes_delivered - bytes delivered for the live streams themselves
  • live_stream - id of the live stream, if available
