2021 Release Notes
Date API Release Notes
2021-07-22 CMS API
  • Add search_syntax field that indicates whether a smart playlist search string uses the v2 and v1 (default) search syntax. Using v2 provides much more flexibility. See Managing Playlists for more information.
2021-07-15 CMS API
  • Add forensic_watermarking field that indicates whether a video has forensic watermarking - see CMS API Reference and Overview: Forensic Watermarking
2021-04-09 CMS API
  • Add sort parameter to get videos in folder - see Working with Folders
