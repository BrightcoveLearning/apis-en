This is a summary of changes to the Live API.

2021-10-01

  • Added cuepoint_id and cancel fields for manual cuepoint insertion.

2019-05-13

  • Added color_full_width to outputs fields for creating a live job.

2019-05-06

  • Added support for adding ad metadata to an ongoing live stream.

2019-01-28

  • Added support for RTMP outputs.

2019-01-21

  • Added support for new input protocols (to allow MPEG2-TS input).
