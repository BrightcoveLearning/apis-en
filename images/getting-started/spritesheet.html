Sprite Sheet

The Roku thumbnails created within the HLS (v7) manifest use the sprite endpoint from the Image API.

The endpoint to generate sprite sheets is:

https://images.brightcovecdn.com/image/v1/{account_id}/video/{video_id}/images/{token}/sprite/{interval}/{width}x{height}/{htiles}x{vtiles}/{sheet}.{format}

The required path parameters are:

URL Params to generate sprite sheets for Image API
Parameter Description
account_id Your Video Cloud account id.
video_id The id of the video selected.
token Your image token.
interval

Tile duration or how often you want a tile; If this is a single image request, the interval will be 1 second.
width x height Determine which measure matches the request; if the requested width & height exceed that of all video assets, then use the asset with the largest dimensions.
htiles x vtiles Vertical and Horizontal images count.
sheet The sheet number you want to get.
format image format ("png", "jpg", "jpeg")
Note that this endpoint does not support the resize, fallback, fillArea, crop, rotate, nocache and watermark query parameters.
To protect the images, make sure the token has only allowed domains of your knowledge.

Limitations

The images extracted from the endpoint are not protected with DRM.
